# 🎓 Classroom Student Attentiveness Detection System

A production-grade, real-time AI system that monitors and analyzes student attentiveness from a live classroom video feed. It detects faces, tracks students, recognizes identity, measures emotions, head pose, and eye state — then computes an attention score displayed on a live dashboard.

## ⚙️ System Architecture

```
Camera/Video
     │
     ▼
[YOLOv8 Face Detector]
     │
     ▼
[DeepSORT Tracker]  ──► assigns persistent track IDs
     │
     ▼
[FaceNet Recognizer] ─► maps track ID → student name
     │
     ├──► [MediaPipe Head Pose] ─► yaw / pitch → forward / away
     ├──► [EAR Blink Detector]  ─► EAR → eyes open / closed
     └──► [FER Emotion]         ─► CNN → emotion label
               │
               ▼
     [Attention Score Calculator]
     Score = 0.35×HeadPose + 0.25×Eye + 0.20×Emotion + 0.20×BodyPose
               │
       ┌───────┴────────┐
       ▼                ▼
  [SQLite DB]     [OpenCV Display]
       │
       ▼
  [FastAPI REST]
       │
       ▼
  [Streamlit Dashboard]
```

## 🧠 Attention Algorithm

```
AttentionScore = 0.35 × HeadPoseScore
               + 0.25 × EyeScore
               + 0.20 × EmotionScore
               + 0.20 × BodyPoseScore
```

| Signal | Value | Score |
|--------|-------|-------|
| Head Pose | Forward | 1.0 |
| Head Pose | Away | 0.0 |
| Eye State | Open | 1.0 |
| Eye State | Closed | 0.0 |
| Emotion | Happy / Neutral | 1.0 |
| Emotion | Surprise | 0.8 |
| Emotion | Sad / Fear | 0.5 |
| Emotion | Angry / Disgust | 0.3–0.4 |
| Body Pose | Upright | 1.0 |
| Body Pose | Slouched | 0.5 |

### Classification Thresholds
| Score Range | Label |
|------------|-------|
| 0.0 – 0.3 | 🔴 Sleeping |
| 0.3 – 0.6 | 🟠 Distracted |
| 0.6 – 1.0 | 🟢 Attentive |

## 🚀 Installation (Step-by-Step)

### Prerequisites
- Python 3.9 or 3.10 (recommended)
- pip
- Webcam or video file
- (Optional) NVIDIA GPU with CUDA for acceleration

### Step 1 — Clone / Extract the Project

```bash
# If downloaded as zip:
cd E:\
unzip student-attentiveness-system.zip
cd student-attentiveness-system
```

### Step 2 — Create a Virtual Environment

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### Step 3 — Install Dependencies

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

> ⚠️ **Note on face_recognition**: This library requires `cmake` and `dlib`. On Windows, install [CMake](https://cmake.org/download/) first, or install pre-built wheels:
> ```bash
> pip install dlib
> pip install face-recognition
> ```

> ⚠️ **TensorFlow / FER**: If you have GPU, install `tensorflow-gpu` instead. On CPU, standard `tensorflow` works fine.

### Step 4 — (Optional) Register Students

Skip this if you just want to test with the simulator.

#### 4a. Collect face images for each student

```bash
python collect_student_data.py --name "Alice Johnson" --count 30
python collect_student_data.py --name "Bob Smith" --count 30
# Repeat for each student
```
- Press **SPACE** to capture each image
- Make sure the student faces the camera in different angles

#### 4b. Generate face encodings

```bash
python -m backend.recognition.dataset_encoder
```
This creates `models/face_encodings.pkl`.

## ▶️ Running the System

Open **three separate terminals** (all with virtualenv activated):

### Terminal 1 — Detection Pipeline (Live Video)

```bash
python backend/main.py
```
- Defaults to webcam (index 0)
- Use a video file: `python backend/main.py --source video.mp4`
- Use IP camera: `python backend/main.py --source rtsp://192.168.1.100/stream`

### Terminal 2 — REST API Backend

```bash
uvicorn backend.api.api:app --reload --host 0.0.0.0 --port 8000
```
API docs: http://localhost:8000/docs

### Terminal 3 — Streamlit Dashboard

```bash
streamlit run dashboard/dashboard.py
```
Dashboard: http://localhost:8501

## ⚡ GPU Acceleration

The system automatically detects CUDA:

```python
# In backend/config.py
DEVICE = "cuda" if torch.cuda.is_available() else "cpu"
```

To verify GPU is being used:
```bash
python -c "import torch; print(torch.cuda.is_available())"
```

For GPU face recognition, change in `config.py`:
```python
FACE_RECOGNITION_MODEL = "cnn"   # GPU-accelerated
```

## ⚙️ Configuration

All settings are in `backend/config.py`:

| Setting | Default | Description |
|---------|---------|-------------|
| `CAMERA_INDEX` | `0` | Webcam index |
| `FRAME_WIDTH/HEIGHT` | `640×480` | Processing resolution |
| `FRAME_SKIP` | `2` | Process every 2nd frame |
| `YOLO_CONF` | `0.40` | Detection confidence |
| `EAR_THRESHOLD` | `0.25` | Eye closure threshold |
| `HEAD_POSE_YAW_THRESHOLD` | `20°` | Max yaw before "away" |
| `FACE_RECOGNITION_TOLERANCE` | `0.50` | Identity match threshold |

## 🔧 Troubleshooting

| Problem | Solution |
|---------|---------|
| `face_recognition` install fails | Install cmake + dlib manually |
| YOLO model not found | It auto-downloads on first run |
| Camera not opening | Change `CAMERA_INDEX` in config.py |
| Dashboard shows no data | Make sure API is running first |
| Slow FPS | Increase `FRAME_SKIP` or use smaller YOLO model |
| MediaPipe errors | `pip install mediapipe --upgrade` |
